FINAL YEAR PROJECT
MUSICAL CONCEPT TECHNICS CLASS BOOKING SYSTEM (MCTCBS)

Skill : PHP

This system consists of three level which are customer, staff and admin. MCTCBS enables customer to view class information including its availability, perform booking of classes, view list of booking that has been made and edit profile. For staff, staff will act as the system administrator where the person is in charge in managing most of all users in MCTCBS. Staff needs to login to the system to access all functionalities for staff. Staff is in charge to manage booking, class, student, teacher and users. For admin, Admin only view booking and manage staff.

  • This system was develop by using bootstrap, HTML5 and CSS for the frontend
  • The backend are using PHP, mysql


System Flow Customer

MCTCBS enables customer to view class information including its availability, perform booking of classes, view list of booking that has been made and edit profile.

After login, customer can select the new book at the top left after concept technics word to go select classes for registration. Customer can view class information and slot left. After made an option then click the book button and customer will go to the next page which is need to fill in student information form registration. customer need to confirm student information again and then submit reservation.

Next, customer need to login first in the online banking login page. After login successful, customer need to confirm the payment and then customer will receive the receipt at the end of the transaction.

Other than that, in the online banking transaction page, customer need to click the "PAY" button to continue to the receipt page. Booking information will be email to customer after the payment complete.

After that, in the receipt page, customer can print out the receipt and also can sent the booking details to customer by whatsapp. Customer can click the "CONTINUE" button to proceed to the next page.

Aside from that, after clicking the "CONTINUE" button in payment receipt page, customer will go directly to list of all booking information that had been made.

Furthermore, customer can edit their profile such as name, phone number and email only.



System Flow for Staff

MCTCBS enables customer to view class information including its availability, perform booking of classes, view list of booking that has been made and edit profile.

Shows that staff can view all booking and can delete booking when the status is unpaid only. Staff also can view the booking status by using filter.

Besides that, staff can view payment report and also can view it using filter by date. It will make easier for staff for viewing the payment report.

Other than that, Staff can manage class and can view student in the class. staff can also manage student details by editing their name, age, and gender.

Next, staff can also manage teacher details.



System Flow for Admin

Admin only view booking and manage staff, Admin can view all of the total users, payment and bookings.

Shows that admin can view all of the booking reports and can filter it by the status of the booking which is paid and unpaid. Admin can view payment report and also can filtered it by date.

Shows that list of classes that admin can view all of the classes and can view student in the class. Furthermore, admin can view all of student details and also can print it out as a report.

Aside from that, admin can manage the staff details and can add a new staff directly. Only admin can manage staff. Besides that, admin can view list of the teacher and their details only.

Lastly, admin can view users that had registered in the system. Admin can view all of the details about the customer details. Figure 4.30 shows list of users which is the customer of the student.
