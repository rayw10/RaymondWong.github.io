FINAL YEAR PROJECT
MUSICAL CONCEPT TECHNICS CLASS BOOKING SYSTEM (MCTCBS)

Skill : PHP

This system consists of three level which are customer, staff and admin. MCTCBS enables customer to view class information including its availability, perform booking of classes, view list of booking that has been made and edit profile. For staff, staff will act as the system administrator where the person is in charge in managing most of all users in MCTCBS. Staff needs to login to the system to access all functionalities for staff. Staff is in charge to manage booking, class, student, teacher and users. For admin, Admin only view booking and manage staff.

  • This system was develop by using bootstrap, HTML5 and CSS for the frontend
  • The backend are using PHP, mysql
